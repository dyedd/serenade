# 人工智能的定义和基础

人工智能（Artificial Intelligence，简称AI）是指通过计算机模拟人类智能的技术。AI 包括机器学习、深度学习、自然语言处理等子领域。

### 示例代码

```python
# 一个简单的机器学习模型
from sklearn.linear_model import LinearRegression

# 创建数据
X = [[1], [2], [3]]
y = [1, 2, 3]

# 训练模型
model = LinearRegression()
model.fit(X, y)

# 预测
pred = model.predict([[4]])
print("Prediction:", pred)
```
